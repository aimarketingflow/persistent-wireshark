Overview

Complete integration of StealthShark network monitoring with NFC WiFi authentication, providing packet capture, Bluetooth scanning, SSH security, and firewall controls.

Prerequisites

# Install dependencies pip3 install PyQt6 psutil # Ensure tshark or tcpdump is available which tshark tcpdump

Basic Usage

Launch GUI Application

cd /Users/flowgirl/Documents/_MobileShield/StealthShark_NFC_Combined python3 stealthshark_nfc_combined.py

Test Packet Capture

# Test monitor functionality python3 test_monitor.py # Test GUI with simulated authentication python3 test_gui.py

Service Management

Install Auto-Start Service

./install_service.sh

Check Service Status

launchctl list | grep com.aimf.stealthshark-nfc

Stop/Start Service

# Stop service launchctl unload ~/Library/LaunchAgents/com.aimf.stealthshark-nfc.plist # Start service launchctl load ~/Library/LaunchAgents/com.aimf.stealthshark-nfc.plist

Uninstall Service

./uninstall_service.sh

Monitoring Features

View Capture Files

# List all captures ls -la ./pcap_captures/session_*/ # Analyze specific capture tcpdump -r ./pcap_captures/session_*/ethernet/*.pcap -nn | head -20

View Logs

# Monitor logs tail -f ./logs/stealthshark-nfc.log # View error logs tail -f ./logs/stealthshark-nfc.error.log # Check monitor logs ls -la ./pcap_captures/logs/

Configuration

Edit Settings

nano config.json

Clear Authentication

rm auth_state.json

Bluetooth Features

View Device History

cat bluetooth_history.json | python3 -m json.tool

Troubleshooting

Permission Issues

sudo chmod +x /usr/sbin/tcpdump

Clear All Data

rm -rf pcap_captures/ logs/ bluetooth_history.json auth_state.json

Development

Run Tests

# Test capture system python3 test_capture.py # Test monitor python3 test_monitor.py # Test GUI python3 test_gui.py

Debug Mode

python3 -u stealthshark_nfc_combined.py 2>&1 | tee debug.log

Key Files

  • stealthshark_nfc_combined.py - Main application
  • persistent_wireshark_monitor.py - Packet capture engine
  • stealthshark_nfc_methods.py - Helper methods
  • config.json - Configuration settings
  • com.aimf.stealthshark-nfc.plist - LaunchAgent service
